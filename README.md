## üöÄ Overview

This Finance Tracker allows users to monitor their income and expenses with a real-time updated balance. The interface follows modern "Soft UI" design trends, featuring smooth transitions, rounded corners, and a clean typography system.

### Key Features
* **Real-time Calculations:** Balance, Income, and Expenses update instantly as transactions are added or removed.
* **Data Persistence:** Uses `LocalStorage` to ensure data remains available even after refreshing the page.
* **Responsive & Modern UI:** A fully responsive "Premium Modern" dashboard built from scratch with CSS.
* **Dynamic List Management:** Add and delete transactions with immediate DOM feedback.
